\begin{frame}
  \frametitle{Nr of house visits per month}

Total number of houses according to the shapefiles = 62998. 
So if we were to believe that \textbf{all} houses were visited each month, then
$$\frac{63932 \, \text{houses}}{20 \, \text{working days}} = 3197\, \text{houses per day},$$

In one day: $$\frac{ 3197\, \text{houses per day}}{8 \frac{\text{working hours per day}}{\text{day}}} =  400\,\text{houses per hour},$$

If we assume that one house visit takes about 15 minutes, then

$$\frac{ 403\, \text{houses per hour}} {4 \, \text{houses per hour per employee}} = 100 \text{ employees}$$

This reasoning would imply that at the height of entomological surveillance (2015-2019), 65 employees were working daily, doing household visits. 

\end{frame}
